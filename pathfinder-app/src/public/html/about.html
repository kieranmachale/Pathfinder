About This Website

Pathfinder is a web application that helps Arduino developers to create, configure, and launch their projects entirely from their web browser. Particular emphasis has been put into the ability to control the Arduino remotely using emerging serial communications tools, enabling developers to change the behaviour of scripts on the fly without needing to modify any code. The website will provide a marketplace of applications, each containing an Arduino script paired with a corresponding ‘serial app’. The script contains code which runs on the Arduino board, while the corresponding serial app contains code that runs in the browser and handles the control of these scripts. The benefit this offers developers is that they can control various aspects of the system’s behaviour, be it hardware or software, without needing to write or re-upload any new code to the Arduino.

The two main goals of the project are firstly, to demonstrate how serial communication can be used to create flexible, malleable Arduino scripts, and secondly, to provide Arduino developers with an online platform where they can quickly prototype their ideas, shortening the gap between conception and power of execution.
